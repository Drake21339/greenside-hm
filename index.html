MARCH-PAWS Assessment

What is this?

MARCH is the life-threatening order: Massive hemorrhage, Airway, Respiration, Circulation, Hypothermia/Head. PAWS is stabilization for evacuation: Pain, Antibiotics, Wounds, Splinting. Order is critical.

How to use it: Tap Complete step on each letter. Select all and only the correct actions for that step to check it off.
